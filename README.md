# HR Attrition Analysis Dashboard (Excel)

## Project Overview

This project analyzes employee attrition using an HR dataset. The dashboard was built entirely in Microsoft Excel using pivot tables, charts, and data cleaning techniques.

## Key Metrics

* Total Employees: 1233
* Attrition Rate: 25%
* Average Age: 39
* Average Salary: 6.5k
* Average Years at Company: 7

## Dashboard Insights

* Research & Development department shows the highest attrition.
* Employees aged 26–35 leave the company more frequently.
* Most employees leaving the company have 0–9 years of experience.
* Entry-level employees contribute the most to attrition.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Data Cleaning
* Data Visualization
